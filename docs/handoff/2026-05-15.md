## Decisions made this session

- Use `@xterm/*` namespace packages (xterm 5.5+), not the old `xterm-addon-*` naming.
- Use `tauri::async_runtime::spawn` (not `tokio::spawn`) for the PTY event forwarder ‚Äî Tauri v2 has its own runtime adapter.
- App-scoped hotkeys (not global) ‚Äî avoids macOS Accessibility prompt on first launch. Decided at planning time; capabilities file reflects this.
- File tree hides dotfiles by default (`show_hidden: false` in `list_directory`). `Cmd+.` toggle deferred to v0.2.
